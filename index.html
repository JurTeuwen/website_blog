WATER -EN VOEDINGSSYSTEEM

Jur en Rens

Ons project is om iets te maken om de plantenbakken in de box te onderhouden. We moeten ervoor zorgen dat de planten voldoende water krijgen. Verder zorgen we ervoor dat je kan zien wat de vochtigheid en de NPK-waarde zijn.

NPK-sensor + arduino + Lcd + solderen

Jur en Rens

25/05/2023

Uiteindelijk hebben we de sensor werkende gekregen op de arduino uno. Ook is het gelukt om informatie door te sturen naar een live server. Hier waren we heel blij mee. Toen kwam het volgende probleem. De NPK-sensor werkt niet op de arduino uno wifi omdat de library die we gebruiken niet werkt op de arduino uno wifi. We hebben geprobeerd de library aan te passen om dit op te lossen maar dit lukte niet meteen en door tijdsgebrek hebben we besloten niet meer met de arduino uno wifi te werken. Toen kwamen we met de oplossing om via een wifimodule het via de arduino uno te doen maar tevergeefs. Dit lukte ook niet. Daarna hebben we besloten om de waarden niet meer op de website te laten zien. Een volgend idee was om de waarden op een app op een telefoon te laten zien. Dit met een bluetooth-module. Dit is ook niet gelukt. Toen kwamen we bij onze laatste optie. De waarden laten zien op een lcd-scherm. Dat lukte natuurlijk wel. We zijn dus bezig geweest met het projecteren van de 4 verschillende waardes op een lcd scherm. Ook hebben we een toevoeging gedaan dat wanneer de waardes nog geladen worden dat er ook daadwerkelijk loading komt te staan op het lcd-scherm. daarnaast zijn we ook begonnen met de code te optimaliseren. We hebben ook het deel van de waarden verwijderd op onze website. Tenslotte hebben we de componenten die vast op de arduino zitten op een printplaat gesoldeerd.

We zijn de laatste maand veel bezig geweest met de NPK-sensor en de Arduino uno wifi. We hebben veel problemen ondervonden. De sensor werkt nog altijd niet. De krijgen de hele tijd een foute waarde terug. Met de Aduino uno wifi lukte het ook niet. We maakte uiteindelijk verbinding met de wifi en ook met de server maar verder kwamen we niet. Ook hebben we in de vakantie ons project mee naar huis mogen nemen. Daar hebben wij er ook veel aan gedaan. Toen we terug op school kwamen kregen we te horen dat we toch een Rasbery Pi mochten gebruiken. De Arduino uno wifi hebben we achter ons gelaten. We hebben één Rasbery Pi gekregen om daarmee conectie te maken met verschillende Arduino's.

NPK-sensor + Website + aduino + app

Jur en Rens

17/03/2023

Er is sinds de laatste blog veel veranderd. We hebben besloten om toch met een Arduino uno wifi te gaan werken. Dit heeft 2 redenen. De eerste reden is de prijs. Een Rasbery Pi is tegenwoordig heel erg duur. Ten tweede hebben we wat problemen met de NPK-sensor. Van deze sensor vinden we geen code voor op de Rasbery Pi en wel voor op een Arduino. Een nadeel van de Arduino is wel dat de waarde van de sensoren veel moeilijker door te sturen zijn naar de server dan bij een Rasbery Pi. We zijn hieraan begonnen maar het is nog niet gelukt. Verder hebben we nog een beetje aan de website gewerkt en zijn we begonnen aan het ontwerpen van aan app op android.
